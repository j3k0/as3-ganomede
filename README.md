# AS3-CLI-Boilerplate

This is a starting point project for small tools written for the command line in actionscript 3.

"Why?" I hear you say. Because.

## Getting started

Install Adobe Air, make sure air tools are in the PATH.

Build
```sh
make build
```

Run
```sh
make run
```

Clean
```sh
make clean
```
